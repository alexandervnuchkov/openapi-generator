В этом разделе вы можете выбрать один из существующих серверов, на который будут отправляться запросы API.

  • Выберите сервер sandbox.dev, если у вас есть аккаунт в песочнице и вы хотите попробовать запросы с тестовыми данными.

  • Выберите рабочий сервер example.dev, если у вас есть аккаунт в рабочей среде и вы хотите попробовать запросы с реальными данными.

Все запросы к API требуют аутентификации.
