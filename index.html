In this section, you can select one of the existing servers which you will use to send the API requests to.

  • Select the sandbox.dev server if you have an account in the sandbox environment and want to try the requests with the testing data.

  • Select the production example.dev server if you have an account in the production environment and want to try the requests with the real data.

All the requests to the API require authentication.
